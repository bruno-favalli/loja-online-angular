Sobre a Minha Loja Online

Bem-vindo à nossa loja! Este projeto foi desenvolvido como parte do projeto final do curso presencial da Caixa com parceria com a plataforma de ensino ADA, com o objetivo de aplicar na prática os conceitos fundamentais e avançados do framework.

Esta é uma loja online totalmente funcional, construída do zero, que consome dados de uma API pública (a Fake Store API) para simular um ambiente de e-commerce real.

Tecnologias Utilizadas:

  • Angular: O framework principal para a construção da interface.
  • TypeScript: A linguagem base que adiciona tipos e segurança ao JavaScript.
  • RxJS: Para gerenciamento de estado reativo e chamadas assíncronas.
  • Angular Router: Para a criação de uma Single Page Application (SPA) com múltiplas páginas.
  • Angular Guards: Para proteger rotas e garantir a segurança da aplicação.
  • HTML & CSS: Para a estrutura e estilização de todos os componentes.

Este projeto demonstra um fluxo completo de uma aplicação de e-commerce, desde a visualização de produtos até um carrinho de compras persistente e autenticação de usuários.
